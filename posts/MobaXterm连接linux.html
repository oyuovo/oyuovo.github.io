前言 ​

使用的版本是Ubuntu22.04，目前只在连接虚拟机尝试成功过，如有问题欢迎指正

前置条件 ​

  1. 安装好MobaXterm官网下载
  2. 在Ubuntu中安装好ssh服务
    shell
    sudo apt install openssh-server
  3. 通过hostname -I可以获取到自己的ip地址

开始连接 ​

  1. 打开MobaXterm的左上角的Session
  2. 选择SSH连接
  3. 在Remote host上填写你上面获得的ip地址
  4. 在Spercify username勾选，然后写上你Ubuntu的用户名
  5. 点击OK，输入密码就可以成功连接了

这里端口默认就行，除非被占用
