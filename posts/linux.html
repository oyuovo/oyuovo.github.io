常用Linux指令-Ubuntu为例 ​

ctrl + alt + t 打开终端

文件和目录操作 ​

  • pwd：显示当前目录路径
  • ls：列出当前目录下的文件和文件夹
    • 选项：
      • -l：依次按照文件权限、链接数、所有者、所属组、文件大小、最后修改时间、文件名的顺序显示详细信息
      • -a：显示所有文件，包括隐藏文件
  • man [要查的命令]：查看命令的使用手册
    • [要查的命令] --help：显示命令的帮助信息(效果类似man)
  • clear：清屏

ctrl + l 也可以清屏

  • cd [目录路径]：切换当前工作目录到指定路径
    • cd ..：返回上一级目录
    • cd ~或者cd：切换到用户主目录
    • cd -：切换到上次所在目录
  • mkdir [目录名]：创建新目录
    • -p：创建多级目录eg. mkdir -p dir1/dir2/dir3
  • rmdir [目录名]：删除空目录
  • cp [源文件路径] [目标文件路径]：复制文件或目录
    • -r：递归复制整个目录及其内容
  • rm [文件路径]：删除文件或目录
    • -r：递归删除目录及其内容
    • -f：强制删除，不提示确认
  • mv [源文件路径] [目标文件路径]：移动或重命名文件或目录
    • mv oldname newname：重命名文件或目录
    • mv file.txt /path/to/destination/：移动文件到指定目录
  • cat [文件路径]：显示文件内容
    • -n：显示行号
  • tail [文件路径]：显示文件的最后几行
    • -n [行数]：指定显示的行数，默认显示最后10行
    • -f：实时跟踪文件内容的变化

    crtl + c 停止跟踪

文本编辑操作 ​

vim文本编辑器 ​

  1. 工作模式
  • 普通模式(Normal Mode)：用于浏览和执行命令（默认模式）
  • 插入模式(Insert Mode)：用于编辑文本，按i或a或o进入插入模式，按Esc返回普通模式
    • i：在光标前插入
    • a：在光标后插入
    • o：在当前行下方新开一行插入
  • 命令模式(Command Mode)：用于保存、退出等操作，按:或者/进入命令模式,按Esc返回普通模式
  • 可视模式(Visual Mode)：用于选择文本，按v进入可视模式，按Esc返回普通模式
  1. 常用命令
  • 进入vim：在终端输入vim [文件名]，如果文件不存在则创建新文件
  • 保存文件：在命令模式下输入:w，然后按Enter
  • 退出vim：在命令模式下输入:q，然后按Enter
  • 保存并退出：在命令模式下输入:wq，然后按Enter
  • 强制退出不保存：在命令模式下输入:q!，然后按Enter

文件权限操作 ​

  • Linux中每个文件都有三组权限：所有者(u)、所属组(g)、其他用户(o)
  • 每组包含三种权限：读(r=4)、写(w=2)、执行(x=1)

使用ls -l查看文件权限时，输出示例：

-rwxr-xr-- 1 user group 1024 Mar 23 10:00 file.txt

第一位-表示普通文件（d为目录），后面每三位一组分别对应所有者、所属组、其他用户的权限

  • chmod [权限] [文件路径]：修改文件权限
    • 数字方式：chmod 755 file.txt（所有者rwx=7，组r-x=5，其他r-x=5）
    • 符号方式：chmod u+x file.txt（给所有者添加执行权限）
    • 常见组合：
      • chmod 777：所有用户可读可写可执行
      • chmod 644：所有者可读写，其他用户只读
      • chmod +x script.sh：给所有用户添加执行权限
    • -R：递归修改目录及其内容的权限
  • chown [用户:组] [文件路径]：修改文件的所有者和所属组
    • chown user file.txt：修改所有者
    • chown user:group file.txt：同时修改所有者和所属组
    • chown :group file.txt：只修改所属组
    • -R：递归修改目录及其内容的所有者
  • chgrp [组名] [文件路径]：修改文件的所属组
    • -R：递归修改

进程管理相关 ​

  • ps：查看当前运行的进程
    • ps aux：查看系统中所有进程的详细信息
    • ps -ef：以全格式显示所有进程（包含父进程ID）
  • top：实时动态查看系统进程和资源使用情况

    按q退出，按M按内存排序，按P按CPU排序

  • kill [进程PID]：终止指定进程
    • kill -9 [PID]：强制终止进程
    • kill -15 [PID]：正常终止进程（默认信号）
  • killall [进程名]：按名称终止所有同名进程
  • jobs：查看当前终端的后台任务
  • bg [任务号]：将暂停的任务放到后台继续运行
  • fg [任务号]：将后台任务调到前台运行

    运行命令时在末尾加&可以直接将任务放到后台运行，如./script.sh &

  • nohup [命令] &：使命令在后台运行，且不受终端关闭影响
  • systemctl [操作] [服务名]：管理系统服务
    • systemctl start nginx：启动服务
    • systemctl stop nginx：停止服务
    • systemctl restart nginx：重启服务
    • systemctl status nginx：查看服务状态
    • systemctl enable nginx：设置开机自启
    • systemctl disable nginx：取消开机自启

压缩归档相关 ​

tar命令 ​

  • tar：用于打包和解包文件（常与压缩工具配合使用）
    • 常用选项：
      • -c：创建归档文件
      • -x：解包归档文件
      • -v：显示操作过程
      • -f：指定归档文件名（必须放在选项最后）
      • -z：使用gzip压缩/解压（.tar.gz 或 .tgz）
      • -j：使用bzip2压缩/解压（.tar.bz2）
    • 常用组合：
      • tar -czvf archive.tar.gz dir/：将目录压缩为.tar.gz
      • tar -xzvf archive.tar.gz：解压.tar.gz文件
      • tar -xzvf archive.tar.gz -C /target/path/：解压到指定目录
      • tar -cjvf archive.tar.bz2 dir/：将目录压缩为.tar.bz2
      • tar -xjvf archive.tar.bz2：解压.tar.bz2文件

zip/unzip命令 ​

  • zip [压缩文件名.zip] [文件...]：压缩文件为.zip格式
    • -r：递归压缩目录，如zip -r archive.zip dir/
  • unzip [压缩文件名.zip]：解压.zip文件
    • -d [目标路径]：解压到指定目录，如unzip archive.zip -d /target/path/
    • -l：查看压缩包内容但不解压

安装软件相关 ​

apt安装 ​

  • 更新软件包列表：sudo apt update
  • 升级已安装的软件包：sudo apt upgrade
  • 安装软件包：sudo apt install [软件包名]
  • 删除软件包：sudo apt remove [软件包名]
  • 搜索软件包：apt search [关键词]
  • 显示已安装的软件包信息：apt show [软件包名]

deb安装 ​

  • 使用dpkg安装.deb包：sudo dpkg -i [包名.deb]
  • 修复依赖关系：sudo apt -f install
  • 卸载.deb包：sudo dpkg -r [包名]
